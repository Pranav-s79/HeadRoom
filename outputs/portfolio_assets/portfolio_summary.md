# Portfolio Summary

ThermalGuard-Cal is a Python research MVP for thermal-aware task scheduling on a
simulated 4x4 many-core chip. It combines a stochastic thermal simulator,
sparse noisy sensors, action-conditioned datasets, point and upper-bound
models, one-sided conformal calibration, scheduler baselines, stress sweeps,
multiseed evaluation, plots, reports, and a Streamlit explanation dashboard.

The project demonstrates how to evaluate thermal scheduler behavior honestly:
not just peak temperature, but hotspot counts, selected-core coverage,
calibration behavior, OOD degradation, and policy-induced feature drift.
