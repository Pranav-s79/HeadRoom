# Recruiter Summary

Built an end-to-end research MVP for thermal-aware scheduling on a simulated
many-core chip. The project includes simulation, dataset generation, model
training, conformal calibration, scheduler evaluation, stress testing,
multiseed summaries, visual reports, and a local Streamlit dashboard for
explaining model decisions and safety metrics.

Key skills shown: Python, scientific computing, ML evaluation, uncertainty
calibration, simulation, data visualization, testing, and careful technical
communication.
